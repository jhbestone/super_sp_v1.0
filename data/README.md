# 广告数据目录

## 数据结构说明

系统需要以下类型的亚马逊广告报告文件（.xlsx 格式）：

### 必需文件

| 文件名 | 说明 | 必需 |
|--------|------|------|
| `商品推广_搜索词_报告_所有_每日_*.xlsx` | 搜索词报告 | ✅ |
| `商品推广_广告位_报告_所有_每日_*.xlsx` | 广告位报告 | 可选 |
| `商品推广_推广的商品_报告_所有_每日_*.xlsx` | 推广商品报告 | 可选 |

### 数据要求

- **时间范围**：至少 30 天数据
- **格式**：Excel (.xlsx)
- **字段要求**：
  - 搜索词报告必须包含：`客户搜索词`、`ASIN`、`花费`、`点击次数`、`广告销售额`、`订单数`
  - 支持中文和英文表头

## 使用方法

1. 从亚马逊广告后台导出报告
2. 将 .xlsx 文件放入此 `data/` 目录
3. 运行 `run.bat` 开始分析

## 注意事项

- 数据文件不会被版本控制追踪（如需追踪请修改 .gitignore）
- 建议每月更新一次数据进行分析
